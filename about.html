About Me

Welcome to my portfolio! I am a passionate and dedicated web developer with a strong foundation in both frontend and backend technologies. My journey in the tech world began with a curiosity to understand how things work on the web, and it has since evolved into a deep commitment to creating seamless, efficient, and scalable web applications.

My Expertise:

I am proficient in the MERN stack (MongoDB, Express, React, Node.js), which allows me to build full-stack applications from scratch. My skills in Java and JavaScript have equipped me with a solid understanding of programming logic and problem-solving techniques, while my knowledge of data structures and algorithms in Java ensures that I can write optimized and efficient code. On the frontend, I specialize in creating dynamic and responsive user interfaces using React, complemented by my proficiency in HTML, CSS, and Bootstrap. My experience with these technologies enables me to design and develop websites that are not only visually appealing but also provide a great user experience. On the backend, I excel in building robust server-side applications using Node.js and Express. My ability to connect these applications to databases, particularly MongoDB, ensures that the data flow between the frontend and backend is smooth and secure. I am also well-versed in the MVC (Model-View-Controller) architecture, which helps me maintain organized and scalable code.

After completing certifications in MERN Stack Development and Data Structures & Algorithms from Coding Ninjas, I began applying my knowledge by working on real-world projects. This hands-on experience not only boosted my confidence but also provided me with valuable insights into how technology works in real-time scenarios. Eventually, I started creating my own projects, which further deepened my understanding of web development.
